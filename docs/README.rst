
ReCorPy Module
=====================


Installation
-------------------

```
pip install recorpy
```

Usage
-------------------



        >>> df = pd.DataFrame(np.random.rand(3,3))
        >>> ReorderCorr(df)


Development setup
-------------------


>>>	$ python3 -m venv env
>>>	$ . env/bin/activate
>>>	$ make deps
>>>	$ tox
